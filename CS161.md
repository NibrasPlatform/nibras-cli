# CS161 (Design and Analysis of Algorithms)

This repo organizes CS161 materials under `Stanford Data/cs161/` with two
categories: `Exams/` and `sections/`. Each item has its own folder and a
`scores.json` file for grading.

## Grading

Each project uses `check` grading. If `grading.json` exists in a project folder,
`nibras` will auto-check answers using one file per question (e.g., `q1.txt`).
If not, it falls back to `scores.json` (manual percentage). You can store
answers in a different folder by passing `--answers-dir`.

`score = earnedPoints / totalPoints * 100`

Auto-check example (`grading.json`):

```json
{
  "totalPoints": 100,
  "questions": [
    {
      "id": "q1",
      "points": 45,
      "answerFile": "q1.txt",
      "solutions": ["Answer A", "Answer B"]
    }
  ]
}
```

Manual grading (`scores.json`):

```json
{
  "earnedPoints": 60,
  "totalPoints": 100
}
```

You can omit `totalPoints` if it is already set in `.nibras.json`.
Validation rules:
- `earnedPoints` must be >= 0
- `totalPoints` must be > 0
- `earnedPoints` must be <= `totalPoints`

## Commands

```bash
nibras cs161 test exam1
nibras cs161 test exam1 --answers-dir /path/to/my/answers
nibras cs161 test section4
nibras cs161 submit exam2
nibras cs161 task exam-final
```

# CS161 (Design and Analysis of Algorithms)

This repo organizes CS161 materials under `Stanford Data/cs161/` with two
categories: `Exams/` and `sections/`. Each item has its own folder and a
`scores.json` file for grading.

## Project IDs

Use these IDs with the CLI:

| ID | Folder |
| --- | --- |
| `exam1` | `Stanford Data/cs161/Exams/1` |
| `exam2` | `Stanford Data/cs161/Exams/2` |
| `exam-final` | `Stanford Data/cs161/Exams/final` |
| `section1` | `Stanford Data/cs161/sections/1` |
| `section2` | `Stanford Data/cs161/sections/2` |
| `section3` | `Stanford Data/cs161/sections/3` |

## Grading

Each project uses `check` grading (percentage):

`score = earnedPoints / totalPoints * 100`

Create or update `scores.json` in the project folder:

```json
{
  "earnedPoints": 60,
  "totalPoints": 100
}
```

You can omit `totalPoints` if it is already set in `.nibras.json`.

## Commands

```bash
nibras cs161 test exam1
nibras cs161 test section4
nibras cs161 submit exam2
nibras cs161 task exam-final
```
